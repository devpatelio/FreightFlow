## EXTRACTION GUIDELINES:

1. **Item Details**: 
   - Extract complete product descriptions including grade, purity, specifications
   - Include packaging information in the description (e.g., "25kg bags")
   - Capture all technical specifications mentioned

2. **Quantities**:
   - Extract exact quantities being shipped
   - Note the unit of measure (pieces, boxes, bags, IBCs, kg, lb, etc.)
   - Calculate totals accurately

3. **Date Handling**:
   - Use ISO format (YYYY-MM-DD) for all dates
   - If packing_date is not specified, use the same as date
   - Order date should reflect when the order was originally placed

4. **Customer Information**:
   - Extract customer ID/account number exactly as shown
   - Match PO number exactly from source document
   - Verify salesperson name spelling

5. **Shipping Addresses**:
   - ship_to: Always extract the primary recipient/destination address
   - shipping_address: Only populate if document explicitly shows a different shipping location
   - Include complete address details (name, street, city, state, zip, country, phone)
   - Look for phrases like "Ship To:", "Deliver To:", "Please Deliver To:", "Shipping Address:"
   - If only one address is mentioned, put it in ship_to and leave shipping_address as null

6. **Verification**:
   - Include "Checked By" name if document shows who verified the packing
   - Leave as null if not specified

7. **Special Notes**:
   - Extract ALL special instructions
   - Include handling requirements (fragile, temperature, orientation)
   - Include delivery requirements or restrictions
   - Include regulatory or compliance notes

8. **Totals Calculation**:
   - total_quantity: Sum of all item quantities
   - total_boxes: Count of total packages/containers being shipped
   - Verify calculations are accurate

## ITEM DESCRIPTION BEST PRACTICES:

When extracting product descriptions, include:
- Product name
- Grade/Type (Industrial, Food Grade, Technical, etc.)
- Purity/Concentration (if applicable)
- Physical form (Liquid, Powder, Granular, etc.)
- Packaging details (Bulk, Bottled, Bagged, etc.)
- Any technical specifications mentioned
- Certifications (if mentioned)

Example good descriptions:
- "Sodium Hydroxide - 50% Solution, Industrial Grade, in IBC containers"
- "Polymer Resin XR-200 - White Powder, 99.5% purity, in 25kg paper bags"
- "Lubricating Oil - SAE 10W-30, Synthetic Blend, in 208L drums"
